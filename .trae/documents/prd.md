# 能耗监测系统 - 产品需求文档

## 1. 产品概述

能耗监测系统是一个面向工业厂区的能源数据可视化管理平台，用于实时监测和分析厂区的用电、用水等能源消耗情况。系统通过直观的数据展示和交互操作，帮助管理人员掌握能源消耗趋势，优化能源使用效率，降低运营成本。

- **目标用户**：工厂管理人员、能源管理专员、设备运维人员
- **核心价值**：实时能耗监控、多维度数据分析、异常预警、能效优化决策支持

## 3. 核心流程

### 3.1 主流程

用户进入系统 → 能源驾驶舱展示整体能耗概况 → 选择分析模块（用电分析/能耗统计）→ 进入具体分析页面 → 查看数据分析结果 → 支持导出和筛选操作

## 4. 用户界面设计

### 4.1 设计风格

- **主题色调**：深色科技风格，主色采用深蓝/深灰背景(#0d1117, #161b22)，辅以青蓝色(#58a6ff)、绿色(#3fb950)作为强调色
- **按钮风格**：圆角按钮，主要操作使用填充色，次要操作使用描边样式
- **字体**：使用现代无衬线字体，标题加粗，数据使用等宽字体显示
- **布局风格**：左侧固定导航菜单 + 顶部二级菜单 + 右侧内容区
- **图标风格**：线性图标，统一线条粗细，使用Lucide图标库

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 全局布局 | 侧边导航 | 深色背景，一级菜单图标+文字，支持折叠/展开，选中高亮效果 |
| 全局布局 | 二级菜单 | 水平排列的Tab标签，位于内容区顶部 |
| 全局布局 | 内容区 | 浅灰色背景，卡片式布局，圆角边框 |
| 能源驾驶舱 | 导航树 | 可折叠树结构，选中高亮，hover效果，仅支持点击查看 |
| 能源驾驶舱 | 统计卡片 | 渐变背景，大号数字显示，同比变化用绿/红色标识涨跌，带图标 |
| 能源驾驶舱 | 图表区域 | 曲线图带网格背景，支持hover显示具体数值，渐变填充效果 |
| 分析页面 | 对比图表 | 双折线/柱状图对比，图例说明，数据标签 |
| 分析页面 | 数据表格 | 表头固定，斑马纹样式，支持排序和筛选 |

### 4.3 响应式设计

- 采用桌面优先设计，主要适配1920x1080及以上分辨率
- 左侧导航可折叠，适应较小屏幕
- 图表自适应容器宽度
- 内容区支持滚动

### 4.4 动画效果

- 页面加载时数据卡片依次淡入，带延迟动画
- 图表绘制有平滑动画效果
- 树节点展开/收起有过渡动画
- 菜单切换有过渡动画
- 数值变化有计数动画效果
